# DevOps

## Overview

This section covers how backend systems are packaged, shipped, deployed, observed, and operated. It focuses on the engineering side of CI/CD, containers, Kubernetes, deployment safety, and monitoring.

## Folder Map

- `fundamentals/`: operational building blocks and deployment concepts
- `interview/`: practical DevOps interview preparation
- `examples/`: applied patterns for Dockerfiles and Kubernetes troubleshooting
