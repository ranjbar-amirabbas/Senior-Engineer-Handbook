# .NET

## Overview

This section covers the core knowledge expected from a senior backend .NET engineer: C# language fundamentals, async and concurrency, dependency injection, ASP.NET Core, Entity Framework, and architecture patterns commonly used in production services.

## Folder Map

- `fundamentals/`: language and runtime concepts that affect code correctness and maintainability
- `framework/`: ASP.NET Core and EF Core guidance grounded in application development
- `architecture/`: design approaches for structuring larger .NET systems
- `interview/`: question-and-answer style preparation
- `examples/`: concrete backend scenarios and implementation patterns
