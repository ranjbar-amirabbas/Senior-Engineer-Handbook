# Interview Topic Template

## Intro

Summarize what this interview topic tests. Mention the difference between a surface-level answer and a senior-level answer.

## Subtopic Name

### Question

Write the interview question exactly as a candidate might hear it.

### Short Answer

Answer in two to four sentences. This should be crisp enough for live interview delivery.

### Deep Explanation

Expand on mechanisms, trade-offs, and examples. Show what changes in production systems.

### Senior-Level Notes

Mention operational concerns, design trade-offs, migration considerations, or scale-related caveats.

### Common Mistakes

- giving a definition with no decision criteria
- ignoring failure modes
- describing framework behavior incorrectly
- overgeneralizing a context-specific best practice
